**🧩 Technologies Used**

* **HTML5** – Provides the structure and semantic layout of the application
* **CSS3** – Handles styling, animations, and responsive design using Flexbox & Grid
* **JavaScript** – Implements calculator logic and user interactions

---

**🛠️ Getting Started**

1. Clone the repository:

   ```bash
   git clone https://github.com/muzaffarvv/calculator.git
   ```

2. Navigate to the project folder:

   ```bash
   cd calculator
   ```

3. **Open `index.html` in your browser**
